# Smart Humidity Monitoring System
A smart humidity monitoring prototype using Arduino Uno R3 and DHT22 sensor with embedded automation logic for environmental monitoring.

## Overview
This project is a smart humidity monitoring and control prototype built using an Arduino Uno R3 and a DHT22 sensor. It is designed to monitor humidity levels and support automated responses when humidity crosses defined thresholds.

## Hardware Used
- Arduino Uno R3
- DHT22 humidity and temperature sensor

## Features
- Reads humidity and temperature data
- Monitors environmental conditions in real time
- Uses threshold-based automation logic
- Supports indoor climate monitoring applications

## Technologies
- Arduino C++
- Python
- Embedded systems
- Sensor integration

## Project Files
- `smart-humidity-monitor.ino` — main Arduino control code
